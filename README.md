# OSTRICH (On-Site Tool for Rapid Image Conversion and Hitfinding)

This is a new generation of the SFX data processing pipeline at SACLA, succeeding [the earlier Cheetah-based pipeine](https://github.com/biochem-fan/cheetah).

## Documentation

Documentation and tutorials will be available in [the wiki](https://github.com/biochem-fan/ostrich/wiki).
